# Product Context — Project Launcher

## What it is

A web application that lets anyone — including non-technical users — launch an AI-driven software project. The user describes what they want to build, sets a deadline, and the app generates a complete GitHub repository with Claude Code agents that develop the project autonomously until it ships.

## Audience

- Non-technical or semi-technical builders who have an idea but don't know where to start
- Developers who want to delegate daily development to AI agents
- Founders and creators who want a product, not just code

## Personality

- **Confident, not arrogant.** The product knows what it's doing. It doesn't over-explain.
- **Calm, not cold.** Reassuring to someone who has never done this before.
- **Modern, not trendy.** Clean and opinionated, not template-looking.
- **Direct.** No marketing fluff. One step at a time. The product does the work.

## Core experience

A 7-step guided wizard that feels like a conversation, not a form. Each step is focused — one question or one action at a time. The user should feel held, not overwhelmed.

## Differentiator

Most AI tools feel generic. This one should feel like it was designed for a specific type of person: someone who has an idea and the ambition to ship it, but needs a system to do it.

## Success metric

The user lands on the site, fills in 4 questions, connects GitHub, and has a working autonomous dev loop set up — all in under 10 minutes.
